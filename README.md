# Dr. Vijay's Dental Clinic Website

A premium, professional, and modern dental clinic website built with **Next.js**, **React**, and **Tailwind CSS**. 

## Features

- **Responsive Design:** Optimized for Desktop, Tablet, and Mobile.
- **Teal Theme:** A clinical, trustworthy, and modern color palette.
- **Smile Gallery:** Showcasing real patient transformations in a clean 2x2 grid.
- **Service Listings:** Detailed descriptions of core dental treatments (Preventive, Implants, Root Canal, etc.).
- **Appointment Booking:** Integrated WhatsApp-based booking system for quick patient conversion.
- **SEO Optimized:** Semantic HTML and meta tags for better search visibility.

## Tech Stack

- **Framework:** Next.js 15 (App Router)
- **Styling:** Tailwind CSS & Vanilla CSS
- **Icons:** Custom SVG Icons
- **Fonts:** Inter (Google Fonts)

## Getting Started

1. **Clone the repository:**
   ```bash
   git clone [repository-url]
   ```

2. **Install dependencies:**
   ```bash
   npm install
   ```

3. **Run the development server:**
   ```bash
   npm run dev
   ```

4. **Build for production:**
   ```bash
   npm run build
   ```

## Folder Structure

- `/app`: Main application routes and logic.
- `/components`: Reusable UI components (Header, Footer, ReviewSlider, etc.).
- `/public/asset`: Image assets, logos, and service photos.
- `globals.css`: Core design system and custom animations.
